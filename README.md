## 🚀 Why this handbook

Most interview prep is either **too shallow** (flashcards you forget by the elevator) or **too academic** (textbooks that never mention what a panel actually asks). This is neither.

It's the resource I wished existed: **practical, visual, and deep** — written so a 2-year backend engineer pivoting to AI *and* a 7-year staff engineer prepping for a foundation-model role both walk away sharper.

## 🖥️ Run it locally

```bash
git clone https://github.com/wowgeekyboy/genai-interview-handbook.git
cd genai-interview-handbook
npm install
npm run dev          # builds /docs and serves at http://localhost:8080
```

The source of truth is a single markdown file — [`GenAI_AI_Engineering_Complete_Handbook.md`](GenAI_AI_Engineering_Complete_Handbook.md). The build script (`build.mjs`) splits it into navigable, searchable pages under `/docs`, which GitHub Pages serves.

```
.
├── GenAI_AI_Engineering_Complete_Handbook.md   # ← the entire handbook (edit here)
├── build.mjs                                   # markdown → site generator
├── serve.mjs                                   # local preview server
├── src/                                        # UI shell (html, css, js, og image)
└── docs/                                        # built site (served by GitHub Pages)
```
